## Sobre

É um software que utiliza algoritmos de Ordenação e busca como o de Ordem topologica e o BFS, pra auxiliar o usuario a organizar suas ideias com o intuito de não perder a linha de raciocinio da execução de suas tarefas/ideias.

## Instalação

Linguagem: Python + JS <br>
Framework: flask
Requisitos: realizar o comando `pip install -r requirements.txt` pra baixar os pre requisitos

## Uso 
Após clonar o repositorio é recomendado, após de instalar o python, criar um ambiente virtual pra instalação dos pré requisitos.
Depois é so rodar `py app.py` e entrar na porta inidicada.
